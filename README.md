# 中文记事本 - Chinese Notepad

一个现代化的单文件中文记事本应用，具备强大的文本编辑和Markdown支持功能。

![应用预览](https://img.shields.io/badge/版本-1.0.0-blue.svg)
![支持语言](https://img.shields.io/badge/语言-中文-red.svg)
![兼容性](https://img.shields.io/badge/兼容性-现代浏览器-green.svg)

## ✨ 主要特性

### 📝 高级文本编辑
- **智能搜索替换** - 支持中英文全词匹配、正则表达式、大小写敏感
- **性能优化** - 大文档搜索缓存，快速导航和滚动
- **撤销/重做** - 最多50步操作历史
- **实时统计** - 字数、行数、字符数、光标位置实时显示
- **自动换行** - 智能响应式文本排版

### 🎨 Markdown支持
- **完整工具栏** - 粗体、斜体、标题、列表、链接、图片等
- **实时格式化** - 浮动工具栏快速插入Markdown语法
- **代码支持** - 行内代码、代码块、数学公式
- **表格和目录** - 自动生成表格和文档目录

### 📁 文件操作
- **多格式保存** - TXT, MD, HTML, CSS, JS, JSON, XML, CSV
- **智能扩展名** - 根据内容自动检测文件类型
- **UTF-8支持** - 完整的中文和Unicode字符支持
- **自动保存** - LocalStorage本地缓存，防止数据丢失

### 🎯 用户体验
- **深色/浅色主题** - 一键切换界面主题
- **全屏模式** - 专注写作环境
- **响应式设计** - 完美适配桌面和移动设备
- **触摸优化** - 移动端友好的交互设计

## 🚀 快速开始

### 在线使用
直接在浏览器中打开 `index.html` 文件即可开始使用。

### 本地运行
```bash
# 使用Python (推荐)
python3 -m http.server 8000

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 📖 使用指南

### 基础操作
- **新建文档** - 点击工具栏"新建"按钮或按 `Ctrl+N`
- **打开文件** - 点击"打开"按钮选择本地文件，或按 `Ctrl+O`
- **保存文件** - 点击"保存"按钮自定义文件名保存，或按 `Ctrl+S`
- **全选文本** - 使用 `Ctrl+A` 快捷键选择全部内容

### 查找替换功能
1. 点击工具栏"查找"按钮或按 `Ctrl+F` 打开查找面板
2. 输入要查找的文本，支持：
   - ✅ 区分大小写搜索
   - ✅ 全词匹配（支持中文）
   - ✅ 正则表达式
3. 使用"查找下一个"/"查找上一个"导航结果
4. 可选择"替换"或"全部替换"功能

### Markdown编辑
- 右侧浮动工具栏提供完整的Markdown格式化功能
- 支持标题、引用、代码块、表格、链接等
- 实时预览语法高亮

### 移动端使用
- 自动适配屏幕尺寸
- 支持触摸手势和操作
- 底部边距可调节，方便输入文本
